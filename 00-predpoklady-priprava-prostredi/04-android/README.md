# **Android Studio: Příprava prostředí**

V poslední sekci přípravy prostředí si nainstalujeme **Android Studio** – oficiální vývojové prostředí pro tvorbu aplikací na systém Android.

## **Upozornění na hardware**

Android Studio je poměrně velký program. Aby se vám s ním pracovalo dobře, potřebujete:

* **RAM:** 8 GB je absolutní minimum, 16 GB je doporučeno.  
* **Disk:** Budete potřebovat cca **10–15 GB volného místa** (IDE + SDK + Emulátory).  
* **Procesor:** Podpora virtualizace (stejné jako u Dockeru a VirtualBoxu).

*Pokud máte velmi starý notebook, dejte vědět vyučujícímu. Aplikace lze testovat i na fyzickém telefonu místo emulátoru, což ušetří hodně výkonu.*
